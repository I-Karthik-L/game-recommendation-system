# game-recommendation-system
A content-based game recommendation system built using TF-IDF and cosine similarity to recommend video games and board games based on tags and descriptions.
## Dataset Information

This project uses two datasets:

1. **Video Game Dataset**
   - Source: Kaggle
   - Link: https://www.kaggle.com/datasets/nikatomashvili/steam-games-dataset
   - Description: Contains video game titles, tags, and descriptions used for content-based recommendations.

2. **Board Game Dataset**
   - File: BGG_Data_Set.csv
   - Description: Board game mechanics and domains used to generate recommendations.
   
Due to GitHub file size limitations, the full video game dataset is not included in this repository.
